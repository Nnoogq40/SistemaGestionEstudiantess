Sistema de Gestión de Estudiantes
Descripción

Aplicación de consola desarrollada en Java para gestionar estudiantes mediante operaciones CRUD (Crear, Buscar, Eliminar y Mostrar).

Funcionalidades
Agregar estudiantes.
Buscar estudiantes por código.
Eliminar estudiantes.
Mostrar todos los estudiantes registrados.
